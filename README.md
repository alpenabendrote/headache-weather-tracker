# Headache Weather Tracker
頭痛の発生および治癒のタイミングと、その時点の気象データ（気圧・温度・湿度・天気）を記録するツールです。

Google Formsからの送信をトリガーとしてGoogle Apps Script (GAS) が動作し、OpenWeatherMap APIから取得したデータをGoogle スプレッドシートに蓄積します。

## 特徴
- **構成:** Google Forms, Google Apps Script, Google Sheetsを使用。
- **データ取得:** OpenWeatherMap API経由で、打刻時点の気象データを取得して追記。（地点は固定）
- **データ蓄積:** データはスプレッドシートに保存されるため、Pythonや各種分析ツールを用いた相関分析に利用可能。

## アーキテクチャ
`[Google Form] --(送信トリガー)--> [Google Apps Script] --(APIリクエスト)--> [OpenWeatherMap]`
`                                           |`
`                                    (データ書き込み)`
`                                           v`
`                                  [Google Spreadsheet]`

## セットアップ手順

### 1. OpenWeatherMapのAPIキー取得
1. [OpenWeatherMap](https://openweathermap.org/) にてアカウントを作成。
2. 管理画面からAPIキー (`API keys`) を発行し、控えておく。

### 2. Googleフォームとスプレッドシートの作成
1. Googleフォームを作成し、状態を記録する質問（例: 「頭痛開始」「完治」を選択するラジオボタン）を設ける。
2. フォームの「回答」タブから、連携するGoogleスプレッドシートを新規作成する。
3. スプレッドシートの1行目に、フォームが生成した列に続けて以下の列名を追加する。
   * `C列: 気圧(hPa)`, `D列: 温度(℃)`, `E列: 湿度(%)`, `F列: 天気`, `G列: 緯度`, `H列: 経度`

### 3. Google Apps Script (GAS) の設定
1. スプレッドシートのメニューから `拡張機能` > `Apps Script` を開く。
2. 本リポジトリの `code.gs` の内容をコピーし、エディタに貼り付けて保存する。
3. プロジェクトの設定（歯車アイコン）を開き、「スクリプト プロパティ」に以下の2つを追加する。
   * プロパティ: `OPENWEATHERMAP_API_KEY` / 値: `手順1で取得したAPIキー`
   * プロパティ: `LOCATION_CITY` / 値: `Tokyo,JP` (対象とする地域)

### 4. トリガーの設定
1. GASエディタのメニューからトリガー（時計アイコン）を開く。
2. 「トリガーを追加」から以下のように設定する。
   * 実行する関数: `onFormSubmit`
   * イベントのソース: `スプレッドシートから`
   * イベントの種類: `フォーム送信時`
3. 保存し、要求されるアクセス権限を承認する。

## 使い方
1. Googleフォームの回答用URLを取得する。
2. スマートフォンのブラウザで該当URLを開き、ホーム画面にショートカットを追加する。
3. 記録時にフォームから送信を行う。バックグラウンドで気象データが付与され、スプレッドシートに記録される。

## セキュリティ・注意事項
- Googleフォームの回答用URLおよびスプレッドシートのURLは、公開リポジトリ等に記載しないよう注意してください。意図しないデータが送信される原因となります。
- APIキー等の認証情報はソースコードに直接記述せず、GASの「スクリプト プロパティ」経由で読み込む仕様としています。
